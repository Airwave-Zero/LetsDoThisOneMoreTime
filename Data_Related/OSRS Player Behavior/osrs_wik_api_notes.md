Normal Acc Lookup:
An alternate URL can be used to return the results in a JSON instead of a CSV, which also includes the corresponding activity name. The URL for this is https://secure.runescape.com/m=hiscore_oldschool/index_lite.json?player=X where X is the player's name.

Ironman Acc Lookup:
index_lite returns a player's rank, level and experience or score in an activity. The URL for index_lite is 
https://secure.runescape.com/m=hiscore_oldschool_ironman/index_lite.json?player=X where X is the player's name.
